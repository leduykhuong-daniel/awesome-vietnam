## Nền tảng & Cộng đồng

- [Viblo](https://viblo.asia) - Nền tảng chia sẻ bài viết kỹ thuật tiếng Việt (blog, tutorial, kinh nghiệm).
- [TopDev](https://topdev.vn) - Cộng đồng và tin tức IT Việt Nam, tuyển dụng, bài viết công nghệ.
- [VNDev](https://www.facebook.com/groups/vn.dev) - Nhóm Facebook cộng đồng developer Việt Nam.
- [WeBuild Curated](https://github.com/webuild-community/curated) - Danh sách dự án open source do người Việt làm (curated list, PR để thêm dự án).
- [WeBuild Community](https://webuild.community/) - Nền tảng kết nối developer Việt Nam: chia sẻ kiến thức, học và build cùng nhau.
- [GDG Vietnam](https://linkedin.com/company/google-developer-groups-vietnam-gdg-vietnam-) - Cộng đồng developers sử dụng công nghệ Google tại Việt Nam.
- [Grokking](https://www.grokking.org/) - Playground cho software engineers trình cao tại Việt Nam.
- [Vietnam Software Internship Project (VSIP)](https://netviet.kr/vsip) - Cộng đồng mentoring, rèn kỹ năng thực tế cho ngành phần mềm (hướng thị trường Hàn Quốc).

## Kỹ sư phần mềm

- [Fullstack Vietnam](https://github.com/awesome-jobs/vietnam) - Danh sách công ty và vị trí tech tại Việt Nam (GitHub awesome-jobs).
- [The Full Snack](https://thefullsnack.com) - Blog kỹ thuật (tiếng Việt) về development, system design.
- [Tôi đi code dạo](https://toidicodedao.com) - Blog về cuộc sống và nghề developer (tiếng Việt).
- [Machine learning cơ bản](https://machinelearningcoban.com/) - Blog ML/CV của anh Tiệp (tiến sĩ Pennsylvania State University).
- [VNHacker](https://vnhacker.blogspot.com/) - Blog security engineering, phân tích bảo mật (anh Thái, Google).
- [Ehkoo](https://ehkoo.com) - Blog JavaScript của nhóm developer Việt ở châu Âu.
- [Codeaholicguy](https://codeaholicguy.com/) - Blog của Hoàng (Shopback, HCMC).
- [FAMI Linux User Group](https://www.familug.org/) - Blog Python, Linux, programming.
- [Kipacast](https://kipacast.info) - Podcast phỏng vấn nhân vật software Việt trên toàn thế giới.
- [Koel](https://github.com/phanan/koel) - Web-based personal audio streaming (Vue + Laravel), tác giả Việt.
- [Octotree](https://github.com/buunguyen/octotree) - Browser extension xem cây thư mục cho GitHub/Gitlab.
- [Ultimate Go](https://github.com/hoanhan101/ultimate-go) - Study guide Go (tác giả Việt).
- [1loc](https://github.com/phuoc-ng/1loc) - Các thao tác JavaScript trong 1 dòng code (phuoc-ng).
- [OpenKey](https://github.com/tuyenvm/OpenKey) - Bộ gõ tiếng Việt (Win/Mac/Linux), fork Unikey.
- [ibus-bamboo](https://github.com/BambooEngine/ibus-bamboo) - Bộ gõ tiếng Việt cho Linux, cập nhật thường xuyên.

## AI & Machine Learning

- [Awesome Vietnamese NLP](https://github.com/vndee/awsome-vietnamese-nlp) - Bộ sưu tập tài nguyên NLP tiếng Việt (corpus, toolkit, model, sentiment).
- [Awesome made by Vietnamese](https://github.com/lampts/awesome-made-by-vietnamese) - Dự án AI/ML do developer Việt Nam thực hiện và đóng góp.
- [Underthesea](https://github.com/undertheseanlp/underthesea) - Vietnamese NLP Toolkit (Python): xử lý tiếng Việt, tokenize, NER, sentiment.
- [Vietnamese NLP corpus](https://github.com/binhvq/news-corpus) - Tài nguyên corpus và NLP tiếng Việt (ví dụ news corpus).
- [VnCoreNLP](https://github.com/vncorenlp/VnCoreNLP) - Vietnamese NLP toolkit (tokenize, POS, NER, dependency parsing).
- [PhoBERT](https://github.com/VinAIResearch/PhoBERT) - Pre-trained language model cho tiếng Việt (VinAI Research).
- [PhoGPT](https://github.com/VinAIResearch/PhoGPT) - LLM generative cho tiếng Việt (VinAI Research).

## Sách & Tài liệu

- [Huyen Chip — Books](https://huyenchip.com/books/) - Trang sách của Huyen Chip (Chip Huyen): *Designing Machine Learning Systems* (O'Reilly), *AI Engineering* (O'Reilly); tác giả gốc Việt, sách có bản tiếng Việt (Nhã Nam).
- [Thiết kế hệ thống học máy (bản tiếng Việt)](https://huyenchip.nhanam.vn/) - Bản dịch tiếng Việt cuốn "Designing Machine Learning Systems" (Nhã Nam).
- [Designing ML Systems — repo](https://github.com/chiphuyen/dmls-book) - Tóm tắt và tài nguyên kèm sách "Designing Machine Learning Systems" (Chip Huyen).
- [Machine Learning Systems Design (booklet)](https://github.com/chiphuyen/machine-learning-systems-design) - Booklet và bài tập thiết kế hệ thống ML (Chip Huyen).
- [Sách dịch / open source tiếng Việt](https://github.com/topics/vietnamese) - GitHub topic: tài liệu, sách tiếng Việt (tìm repo dịch sách kỹ thuật).
